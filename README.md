### 1. What is my Project name?
Ans: My project name is keen-keeper.

### 2. A short description in my Project?
Ans: KeenKeeper is a responsive web app that helps users track and maintain meaningful relationships by managing friend interactions, reminders, and communication history.

### 3. What technology did i use in this project?
Ans: Html, Css, Tailwind CSS, React.js, React Router DOM, Daisy Ui.

### 4. 3 key features describ of my project?
Ans: 
  1. Smart Friend Tracking System
Easily manage and monitor your relationships by tracking the number of days since last contact, status (overdue, almost due, on-track), and personalized goals for each friend.

  2. Interactive Timeline with Real-Time Updates
Log interactions (Call, Text, Video) directly from the Friend Details page. Each action instantly updates the timeline and shows a toast notification for better user feedback.

  3. Visual Analytics Dashboard
Get insights into your communication habits through a dynamic pie chart that displays the distribution of calls, texts, and video interactions.
